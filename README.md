# BeamNGGuesser
A browser game similar to GeoGuessr and GTA V's GTAGuessr where you guess locations on the Italy map in BeamNG.drive based on pictures you are given.

# Open game.html to play!

### Add more locations:
If you want to add more locations, open the 6000x6000 map in any image editing software, and get the pixel coordinates. Use the included Python program coords_converter.py to convert the numbers into the correct format. Put the output into the locations.js file.
